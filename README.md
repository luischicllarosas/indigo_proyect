# indigoapp
## NOTA : Este proyecto no incluye las credenciales de la base de datos por proteccion

## Instalar las dependencias antes de iniciar : 
```
npm install
```

### Iniciar servidor virtual para ver el proyecto
```
npm run serve
```

### Compilar para produccion
```
npm run build
```
